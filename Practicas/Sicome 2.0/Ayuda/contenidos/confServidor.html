Parámetros del servidor

En la pestaña de configuración de la conexión existen varios parámetros configurables, como se puede apreciar en la siguiente figura:

Los parámetros que realmente hay que definir son el nombre del host y el puerto de conexión. El nombre del host se ha de definir mediante el nombre del host, seguido del dominio donde se encuentre, separados ambos por un punto. Por ejemplo: lucano.uco.es.

Una vez escrito el nombre del host, para pasar al siguiente campo de parámetro basta con pulsar el tabulador. Al pasar al campo de dirección IP puede que tarde un poco pues ha de buscar la IP del host escrito para mostrarla en dicho campo. De esto se desprende que dicho campo no ha de rellenarse. Si la dirección IP aparece con éxito significa que el host se ha encontrado y es correcto. De lo contrario aparecerá un cuadro de diálogo expresando que el host es desconocido. En este caso hay que cumplimentar de nuevo el campo nombre de host. Este proceso se repite hasta que se inserte un host correcto.

Por último, hay que establecer el puerto de comunicación que se usará para realizar las peticiones y recibir los servicios por parte del servidor. La información sobre el nombre del host y el puerto será facilitada por el profesor de la asignatura pertinente o bien se encontrará disponible en la página web del simulador.
